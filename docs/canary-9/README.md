# canary-9 Documentation Index

This directory contains documentation for the KiloCode canary.9 and canary.10 releases, covering the settings system overhaul, MAOS agent infrastructure, webview architecture, and performance work.

## How to use these docs

- Docs are generated by AI agents to help other AI agents and developers understand the codebase.
- SVG diagrams are inline in markdown and render on GitHub — no external viewer needed.
- The `agent-reference/` directory contains the main architecture docs, updated for canary.9.
