# 25 — Evidence Ledger Automation

> **ABSOLUTE RULE — READ-ONLY ON UPSTREAM**
> No `git push`, no `git merge`, no `git rebase`, no `git reset --hard` to upstream branches. No mutation of `main`/`master`/`integration/main` of any repo without explicit operator quorum approval (see doc 04). All automation in this document is **read-only against repository working trees** and writes only into the evidence root. If a script in this document appears to mutate a repo, it is a bug — stop and report.

## Goal

Doc 13 (`13_EVIDENCE_LEDGER_TEMPLATES.md`) defines the markdown shape of every ledger we must produce during the rescue campaign. Manually filling these tables across 11 ecosystem repos × 9 phases is high-volume, low-signal, and error-prone.

This document provides an automation layer that:

1. **Reads** every repo with read-only Git plumbing commands.
2. **Writes** populated ledger sections into `<EVIDENCE_ROOT>/`.
3. **Re-runs safely** — append-only logs, never overwrites.
4. **Stays inside the evidence root** — repos themselves are never touched.

Cross-references: doc 13 (templates), doc 04 (9-phase plan), doc 05 (repo inventory), doc 09 (secret-scan patterns), doc 16 (11-repo enumeration).

## Bash automation — `populate_ledgers.sh`

Save as `<EVIDENCE_ROOT>/scripts/populate_ledgers.sh` and `chmod +x`.

```bash
#!/usr/bin/env bash
# populate_ledgers.sh
# Usage: populate_ledgers.sh <evidence_root> <repo_path>...
# Read-only against repos. Append-only ledger sections. Safe to re-run.

set -euo pipefail

if [ "$#" -lt 2 ]; then
  echo "Usage: $0 <evidence_root> <repo_path> [<repo_path>...]" >&2
  exit 64
fi

EVID="$1"; shift
STAMP="$(date -u +%Y-%m-%dT%H:%M:%SZ)"

mkdir -p "$EVID/raw_per_repo" "$EVID/scripts"

INV="$EVID/repo_inventory.md"
UPS="$EVID/upstream_remote_proof.md"
SEC="$EVID/security_scan_log.md"

init_ledger() {
  local f="$1" title="$2"
  if [ ! -f "$f" ]; then
    {
      echo "# $title"
      echo ""
      echo "> Append-only ledger. Each run appends a new \`## Run\` section."
      echo "> Generated by populate_ledgers.sh — do not hand-edit existing run sections."
      echo ""
    } > "$f"
  fi
}

init_ledger "$INV" "Repo Inventory Ledger"
init_ledger "$UPS" "Upstream Remote Proof Ledger"
init_ledger "$SEC" "Security Scan Log"

atomic_append() {
  local target="$1" content_file="$2"
  local tmp; tmp="$(mktemp "${target}.XXXXXX")"
  cat "$target" "$content_file" > "$tmp"
  mv "$tmp" "$target"
}

git_capture() { git -C "$1" "${@:2}" 2>&1 || true; }

scan_secrets() {
  local repo="$1" out="$2"
  local patterns=(
    'AKIA[0-9A-Z]{16}'
    'AIza[0-9A-Za-z_\-]{35}'
    'ghp_[0-9A-Za-z]{36,}'
    'gho_[0-9A-Za-z]{36,}'
    'ghs_[0-9A-Za-z]{36,}'
    'sk-[A-Za-z0-9]{20,}'
    'xox[baprs]-[0-9A-Za-z\-]{10,}'
    '-----BEGIN [A-Z ]*PRIVATE KEY-----'
    'PRIVATE_KEY'
    'API_SECRET'
    'CLIENT_SECRET'
    'AWS_SECRET_ACCESS_KEY'
  )
  : > "$out"
  for pat in "${patterns[@]}"; do
    grep -nIrE \
      --exclude-dir=.git --exclude-dir=node_modules \
      --exclude-dir=dist --exclude-dir=build \
      --exclude-dir=.venv --exclude-dir=__pycache__ \
      "$pat" "$repo" 2>/dev/null \
      | awk -F: -v p="$pat" '{print $1 ":" $2 "  pattern=" p}' \
      >> "$out" || true
  done
}

for repo in "$@"; do
  if [ ! -d "$repo/.git" ]; then
    echo "skip: $repo is not a git repo" >&2
    continue
  fi
  repo_name="$(basename "$repo")"
  raw_run="$EVID/raw_per_repo/$repo_name/$STAMP"
  mkdir -p "$raw_run"

  git_capture "$repo" rev-parse HEAD              > "$raw_run/HEAD.txt"
  git_capture "$repo" rev-parse --abbrev-ref HEAD > "$raw_run/branch.txt"
  git_capture "$repo" remote -v                   > "$raw_run/remote_v.txt"
  git_capture "$repo" branch -avv                 > "$raw_run/branch_avv.txt"
  git_capture "$repo" status --short --branch     > "$raw_run/status.txt"
  git_capture "$repo" log --oneline -n 80         > "$raw_run/log.txt"
  git_capture "$repo" reflog --date=iso -n 80     > "$raw_run/reflog.txt"

  div_upstream="$(git -C "$repo" rev-list --left-right --count upstream/main...HEAD 2>/dev/null || echo 'n/a')"
  div_origin="$(git -C "$repo"   rev-list --left-right --count origin/main...HEAD   2>/dev/null || echo 'n/a')"

  HEAD_SHA="$(cat "$raw_run/HEAD.txt")"
  CUR_BRANCH="$(cat "$raw_run/branch.txt")"

  inv_tmp="$(mktemp)"
  {
    echo ""
    echo "## Run $STAMP — $repo_name"
    echo ""
    echo "- **Path:** \`$repo\`"
    echo "- **HEAD:** \`$HEAD_SHA\`"
    echo "- **Branch:** \`$CUR_BRANCH\`"
    echo "- **Divergence vs upstream/main:** \`$div_upstream\`"
    echo "- **Divergence vs origin/main:** \`$div_origin\`"
    echo ""
    echo '<details><summary>status</summary>'; echo ''
    echo '```'; cat "$raw_run/status.txt"; echo '```'
    echo ''; echo '</details>'; echo ''
  } > "$inv_tmp"
  atomic_append "$INV" "$inv_tmp"
  rm -f "$inv_tmp"

  ups_tmp="$(mktemp)"
  {
    echo ""
    echo "## Run $STAMP — $repo_name"
    echo ""
    echo '### remote -v'; echo ''
    echo '```'; cat "$raw_run/remote_v.txt"; echo '```'
    echo ''
  } > "$ups_tmp"
  atomic_append "$UPS" "$ups_tmp"
  rm -f "$ups_tmp"

  scan_out="$raw_run/secret_scan.txt"
  scan_secrets "$repo" "$scan_out"
  scan_count="$(wc -l < "$scan_out" | tr -d ' ')"

  sec_tmp="$(mktemp)"
  {
    echo ""
    echo "## Run $STAMP — $repo_name"
    echo "- **Hit count:** \`$scan_count\` (REDACTED — only path:line emitted)"
    if [ "$scan_count" = "0" ]; then
      echo "- Result: **PASS**"
    else
      echo "- Result: **REVIEW REQUIRED**"
      echo ''; echo '```'; cat "$scan_out"; echo '```'
    fi
    echo ''
  } > "$sec_tmp"
  atomic_append "$SEC" "$sec_tmp"
  rm -f "$sec_tmp"

  echo "ok: $repo_name (HEAD=$HEAD_SHA scan_hits=$scan_count)" >&2
done

echo "" >&2
echo "Ledgers updated under: $EVID" >&2
```

## Backup ledger — `populate_backup_log.sh`

```bash
#!/usr/bin/env bash
# Verifies Phase 1 backup artifacts; with --restore-test, clones each mirror and verifies.
set -euo pipefail
EVID="$1"; shift
RESTORE_TEST=0
[ "${1:-}" = "--restore-test" ] && { RESTORE_TEST=1; shift; }
STAMP="$(date -u +%Y-%m-%dT%H:%M:%SZ)"
LEDGER="$EVID/backup_restore_proof.md"
[ -f "$LEDGER" ] || { echo "# Backup & Restore Proof Ledger"; echo "> Append-only."; echo ""; } > "$LEDGER"

section_tmp="$(mktemp)"
{
  echo ""; echo "## Run $STAMP"
  echo ""; echo "Mode: $([ "$RESTORE_TEST" = 1 ] && echo 'restore-test' || echo 'inventory')"
  echo ""
  echo "| Repo | Mirror HEAD | Bundle bytes | Bundle sha256 | Restore-test |"
  echo "|---|---|---|---|---|"
} > "$section_tmp"

shopt -s nullglob
for mirror in "$EVID/mirrors/"*.mirror.git; do
  name="$(basename "$mirror" .mirror.git)"
  bundle="$EVID/bundles/${name}.bundle"
  mhead="$(git --git-dir "$mirror" rev-parse HEAD 2>/dev/null || echo 'n/a')"
  if [ -f "$bundle" ]; then
    bsize="$(stat -c%s "$bundle" 2>/dev/null || stat -f%z "$bundle")"
    bsha="$(sha256sum "$bundle" | awk '{print $1}')"
  else
    bsize=missing; bsha=missing
  fi
  rstatus=skipped
  if [ "$RESTORE_TEST" = 1 ]; then
    tmpdir="$(mktemp -d)"
    if git clone --quiet "$mirror" "$tmpdir/clone" >/dev/null 2>&1; then
      chead="$(git -C "$tmpdir/clone" rev-parse HEAD 2>/dev/null || echo 'n/a')"
      [ "$chead" = "$mhead" ] && rstatus=PASS || rstatus="FAIL ($chead vs $mhead)"
    else rstatus="FAIL (clone error)"; fi
    rm -rf "$tmpdir"
  fi
  echo "| $name | \`$mhead\` | $bsize | \`$bsha\` | $rstatus |" >> "$section_tmp"
done
echo "" >> "$section_tmp"

tmp="$(mktemp "${LEDGER}.XXXXXX")"
cat "$LEDGER" "$section_tmp" > "$tmp"
mv "$tmp" "$LEDGER"
rm -f "$section_tmp"
echo "Backup ledger: $LEDGER" >&2
```

## Read-only guarantee

Every script touches repos with these commands only:
- `git rev-parse`, `git status`, `git log`, `git remote -v`, `git branch -avv`, `git reflog`, `git diff`, `git rev-list --left-right --count`, `git merge-base`, `git for-each-ref` (mirrors only).

Explicitly excluded:
- No `git fetch` (Phase 2 concern, separate ledger).
- No `git fsck`, `git gc`, `git prune`, `git pack-refs`, `git repack`.
- No write to `.git/` of source repos. No write to working trees. No `git config` changes.

`populate_backup_log.sh` performs `git clone` of mirror artifacts under `--restore-test`, but those are reads of artifacts already in `<EVIDENCE_ROOT>/mirrors/`, into temp dirs that are removed.

## Idempotency + race-safety

- Atomic append: every update is `mktemp` + `cat` + `mv`. Readers never see partial sections.
- Timestamp + repo path differentiates concurrent runs.
- No truncation. Re-runs are non-destructive.
- All timestamps are UTC (`date -u`).

## What automation does NOT replace

- Operator judgment on which feature cards to create, scope assignment, conflict detection.
- Quorum approval (Owner + Verifier + Challenger) before Phase 7 actions.
- Triage of secret-scan hits — REDACTED-by-default means operator inspects path:line in private session.
- Decision to fetch upstream — separate Phase 2 step requires explicit approval.
