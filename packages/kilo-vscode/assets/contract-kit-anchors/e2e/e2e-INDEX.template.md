## How to review a trace

1. Click the **trace** link above for any failing test.
2. Download `trace.zip` (Playwright) or watch the `.mp4` (Cypress).
3. For Playwright: open at <https://trace.playwright.dev/> or run
   `npx playwright show-trace trace.zip` locally.
4. For Cypress: the `.mp4` is the full session; Test Replay is at
   `cloud.cypress.io` if `CYPRESS_RECORD_KEY` is configured.

## Sign-off (Anchor 3)

Non-coders sign off each acceptance criterion only **after watching the
trace**. The Contract Kit panel writes a JSON record to
`verification/sign-offs/{criterionId}.json` containing
`{ approvedBy, approvedAt, traceUrl }`. The
`proof.user-acceptance-signed` gate fails until every criterion has a
sign-off file.

## Visual regression baselines
